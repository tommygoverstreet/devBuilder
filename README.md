# DevBuilder

DevBuilder is a comprehensive suite of web-based developer tools designed to speed up your workflow. It includes visual builders and generators for various languages and frameworks.

## Features

* **SQL Builder**: Visually construct complex SQL queries.
* **PHP Component Creator**: Generate standardized PHP classes, interfaces, and traits.
* **React Builder**: Scaffold React components with hooks and TypeScript support.
* **HTML/CSS Builders**: Visual generators for semantic HTML and CSS styles.
* **Markdown Editor**: Real-time Markdown preview and export.
* **Mermaid Builder**: Create diagrams and charts using text.
* **JSON Tools**: Format, minify, and validate JSON data.
* **Site Page Generator**: Assemble full HTML pages from pre-built sections.
* **UI Blocks**: Ready-to-use HTML/CSS components (Bootstrap, Tailwind, Plain CSS).

## Deployment

### Netlify

This project is ready for deployment on Netlify.

1. Connect your GitHub repository to Netlify.
2. Set the **Publish directory** to `.` (root).
3. No build command is required as this is a static site.

### Local Development

1. Clone the repository.
2. Open `index.html` in your browser or serve with a local server (e.g., Live Server in VS Code).
